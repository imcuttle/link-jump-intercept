> 拦截浏览器跳转，包括点击 a 标签，window.open 方法；  
> 一般可以用于拦截测试环境、线上环境互相跳转

## Installation

```bash
npm install link-jump-intercept
# or use yarn
yarn add link-jump-intercept
```

## Usage

```javascript
import linkJumpIntercept from 'link-jump-intercept'
```
